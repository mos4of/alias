# Alias Game - Telegram Bot with Mini App

Telegram-бот с Mini App для игры в Alias (Элиас) на JavaScript.

## Функционал

- `/start` → кнопка «Играть»
- запуск Mini App
- выбор уровня сложности: легкий / средний / сложный
- таймер (60 сек)
- очки
- кнопки: «Угадали», «Пропустить»
- смена слов
- смена игрока

## Структура проекта

```
alias-bot/
├── bot.js              # Основной файл бота
├── words.js            # База слов для игры
├── package.json        # Зависимости и скрипты
├── README.md           # Эта инструкция
└── public/             # Файлы Mini App
    ├── index.html      # HTML-разметка
    ├── style.css       # Стили
    └── app.js          # Логика Mini App
```

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd alias-bot
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Создайте файл `.env` в корне проекта и добавьте туда:
   ```
   BOT_TOKEN=your_telegram_bot_token_here
   WEB_APP_URL=https://your-domain.com/  # URL где будет размещен Mini App
   ```

4. Запустите бота:
   ```bash
   npm start
   ```

   Для разработки с автоматической перезагрузкой:
   ```bash
   npm run dev
   ```

## Деплой

Для работы Mini App необходимо разместить файлы из папки `public/` на веб-сервере, доступном по HTTPS (требование Telegram).

### Варианты деплоя:

1. **GitHub Pages** (бесплатно для статических файлов):
   - Запушьте код в репозиторий
   - В настройках репозитория включите GitHub Pages для ветки `main` и папки `/public`
   - Получите URL вида `https://username.github.io/repository/`

2. **Vercel** (бесплатно):
   - Установите Vercel CLI: `npm i -g vercel`
   - Выполните `vercel` в папке проекта и следуйте инструкциям

3. **Netlify** (бесплатно):
   - Перетащите папку `public` в панель Netlify Drop
   - Или подключите репозиторий к Netlify

4. **Собственный сервер**:
   - Разместите содержимое папки `public/` на вашем веб-сервере с HTTPS

## Важные замечания

1. Для работы Mini App в Telegram необходимо использовать HTTPS-соединение
2. Локальная разработка возможна с использованием ngrok или аналогичных инструментов для создания HTTPS-туннеля
3. При первом запуске бота вам нужно будет получить токен у @BotFather в Telegram
4. Mini App будет доступен по кнопке "Открыть игру" после выбора уровня сложности

## Переменные окружения

- `BOT_TOKEN` - токен вашего Telegram бота (обязательно)
- `WEB_APP_URL` - URL где размещен Mini App (обязательно для работы web_app кнопки)

## Как играть

1. Начните диалог с ботом командой `/start`
2. Нажмите кнопку «Играть»
3. Выберите уровень сложности
4. Нажмите «Открыть игру» для запуска Mini App
5. В Mini App покажется слово и действие, которое нужно изобразить
6. Команда должна угадать слово за 60 секунд
7. Нажимайте «Угадали» если слово угадано, или «Пропустить» если нет
8. После окончания времени покажется итоговый счет

## Технологии

- [Telegraf.js](https://telegraf.js.org/) - фреймворк для Telegram ботов
- Telegram WebApp API - для Mini App
- HTML5/CSS3/JavaScript - для клиентской части
- Node.js - серверная платформа
